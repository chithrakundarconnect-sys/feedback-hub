# **App Name**: Frontend Framework

## Core Features:

- Base Layout Component: Create a global layout component for consistent UI across all pages.
- Dynamic Route Handling: Implement basic route handling using Next.js App Router for multiple pages.
- Page Transitions: Use generative AI tool that will make available reusable functions to handle simple animated transitions when switching pages, enhancing user experience.
- Theme Management: Set up basic theme support (light/dark) using CSS variables and React context, for ease of managing theme preferences later.
- Global Styles: Establish a global stylesheet in /app, that uses modern CSS techniques (like CSS modules or styled-components) for styling.

## Style Guidelines:

- Primary color: Soft blue (#A0BFE0) to represent professionalism and calmness.
- Background color: Light gray (#F0F2F5) for a clean, modern backdrop.
- Accent color: Muted purple (#BCA0E0) for interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable design.
- Use simple, consistent line icons from a library like Feather or Phosphor.
- Design a clean, spacious layout using a 12-column grid system.
- Implement subtle animations, like fade-in transitions or hover effects, to enhance user experience.
