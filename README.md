#About Project

Feedback Hub allows customers to submit feedback and the admin to view and manage it in a secure dashboard.
All feedback updates in real-time using Firebase Firestore.

This project was built to demonstrate full-stack development using modern technologies.

---

#Features

User

- Submit feedback message
- Optional screenshot attachment
- Stored in database instantly

Admin

- Secure admin login
- View all feedback
- Real-time dashboard
- Delete feedback
- Protected admin page

---

#Tech Stack

- Next.js (App Router)
- Firebase Authentication
- Firestore Database
- Tailwind CSS
- TypeScript
- Git & GitHub
- Vercel (Deployment)

---

#How It Works

1. User submits feedback
2. Feedback saved to Firebase Firestore
3. Admin logs into dashboard
4. Admin can view and delete feedback
5. Updates appear instantly (real-time)
